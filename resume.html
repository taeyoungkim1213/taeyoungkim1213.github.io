신규 기능 개발 · 기존 시스템 고도화 · 운영 안정화를 담당해온 Java/Spring 기반 백엔드 개발자입니다.

예약·결제·설문·알림·통계·CMS 등 실사용자가 많은 핵심 기능을 직접 설계·구현·운영했으며,
외부 API 연동, 배치/스케줄러, 보안(CSAP) 인증까지 서비스 전반의 품질과 안정성을 책임지는 역할을 수행해왔습니다.

핵심 역량 · 핵심 성과

핵심 역량
  • 트랜잭션을 기반으로 SELECT FOR UPDATE DB 비관적 락 적용 → 데이터 정합성 확보
  • 데드락 발생 상황을 고려한 백오프 기반 재시도 로직 구현
  • 외부 시스템(PG, 인증, 알림, 화상 API) 연동 및 장애 대응 경험
  • 공통 모듈 설계 및 RESTful API 제공으로 서비스 확장성 향상
  • CODE-RAY 기반 취약점 점검 및 시큐어 코딩 적용으로 CSAP 인증 대응
핵심 성과
  • 동시 요청 환경에서도 데이터 충돌 없이 안정적인 처리 구조 구축
  • 엔드투엔드 서비스 설계를 통한 운영 안정성 향상
  • 설문 기능 공통화로 신규 서비스 연계 속도 개선
  • 웹·모바일 알림 통합으로 사용자 공지 전달 신뢰도 향상

에이아이데이타(데이타이음) · 대규모 사용자 트래픽 환경의 체험예약 플랫폼
핵심 기여
피크 시간대에 동시 예약으로 인해 오버부킹 현상 발생 트랜잭션과 락 전략을 재설계하여 정합성 문제 해결
문제 상황
예약시 잔여 좌석 수량 조회 후 처리되는 구조에서 트랜잭션 간 간섭이 발생하며 중복 처리 및 데이터 무결성 훼손 위험 존재
해결 방법
  • 충돌 빈도가 높은 환경이라 판단하여 낙관적 락 대신 비관적 락 선택
  • SELECT FOR UPDATE로 핵심 자원 행 잠금 적용 → 선점 트랜잭션 완료 전 접근 차단
  • 트랜잭션 범위를 검증부터 데이터 생성까지 확장하여 정합성 보장
  • 데드락 발생 시 즉시 실패하지 않도록 백오프 기반 재시도 로직 구현
성과
  • 피크 시간대 동시 요청 환경에서도 데이터 충돌 0건 달성
  • TPS 증가 상황에서도 안정적인 처리 구조 확보
  • 동시성 이슈 해결 경험 확보
