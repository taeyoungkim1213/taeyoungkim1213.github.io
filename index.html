체험 프로그램 예약/연수/결제 운영 시스템
프로젝트 소개
경상남도교육청이 운영하는 ‘미래교육원’은 학생·교직원·학부모 대상의 체험 프로그램 예약, 연수 관리, 유료 결제까지 지원하는 통합 플랫폼입니다.
예약 신청부터 결제, 알림톡 발송, QR코드 기반 현장 참여 확인 및 퀴즈/점수 부여까지 교육 체험 전 과정을 디지털화하여 운영 효율성과 사용자 편의성을 높이고 있습니다.
주요 업무
• 체험 프로그램 예약/결제 트랜잭션 로직 설계 및 동시성 제어 구현
• 결제 모듈을 TOSS PG로 연동하고 결제 검증·콜백 처리 강화
• 예약 완료 및 취소 시 비즈톡 알림톡 발송 기능 개발
• QR코드 기반 퀴즈 점수 부여 및 현장 이벤트 처리 기능 개발
• Quartz를 활용한 휴면 계정 자동 삭제 구현
• 시스템 전반 유지보수 및 장애 대응 참여 중
문제/해결/성과

문제

피크 시간대 동시 예약 처리에서 좌석이 초과(오버부킹)되고 간헐적 데드락으로 예약 지연이 발생했습니다.

해결

• DB 행 잠금(SELECT ... FOR UPDATE)을 적용해 좌석을 동시 잠금하고, 데드락 발생 시 백오프(backoff) 기반으로 재시도(최대 N회)하도록 재시도 로직을 구현하여 트랜잭션을 안정적으로 재실행하도록 했습니다.

→ 동시 예약 환경에서 비관적 락(FOR UPDATE)이 낙관적 락보다 적합하다고 판단해 채택했습니다.

성과

• 오버부킹을 제거하여 동시예약 시 좌석 초과 0건을 달성했습니다.
• 데드락 발생 상황에서도 예약이 안정적으로 처리되도록 만들어 서비스 신뢰도를 높였습니다.
• 피크 시간대(최대 동시 200명 접속)에도 예약 처리 성공률 100%를 유지했습니다.
• TOSS 연동과 비즈톡 알림 도입으로 결제 및 예약 안내 UX를 개선했습니다.
기술 스택
Java, Spring, MyBatis, JSP, MariaDB, Quartz, TOSS PG, Biztalk

• 자주 쓰이는 검색에 인덱스를 추가하고 중복 조회를 줄여 조회 속도를 개선했습니다.

→ 사용자의 빠른 예약 확인을 위해 UI 개선 + DB 튜닝을 동시에 적용했습니다.

성과

→ SDK 인증 문제는 멀티 인스턴스 환경에서 발생했기 때문에, 단일 초기화 전략을 선택했습니다.

성과

• 웹·모바일 간 실시간 알림 동기화를 달성하여 공지 전달 신뢰도를 회복했습니다.
• 외부기관 연동 설문 API를 안정화하여 시스템 활용도를 높였습니다.
• 모바일 알림 실패율을 15% → 0%로 개선했습니다.
