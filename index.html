충청북도교육청 다채움 플랫폼 2024.01 ~ 2025.09 · 9명

㈜데이타이음 · AI 기반 교수·학습 통합 플랫폼
프로젝트 소개
충청북도교육청이 추진한 ‘다채움’은 초·중·고 학생의 학습 이력, 포트폴리오, 비인지적 요소(자기조절학습 등)를 통합 관리하고 AI·빅데이터 기반으로 학습 수준 진단과 맞춤형 학습을 지원하는 학생 성장 지원 플랫폼입니다.
교사·학생·학부모 모두가 참여하는 통합 서비스로, 수업 설계 공유, 형성평가, 독서 활동 관리, 전자책 서비스, 설문 및 피드백 기능 등을 제공하며 현재 교육청 및 산하기관에서 실제 운영 중입니다.
주요 업무
• 관리자 페이지에서 회원 관리, 설문조사 관리, 시스템 알림, 게시판(FAQ·공지사항) 기능을 개발
• 회원가입 시 개인정보 암호화 및 비밀번호 초기화 기능을 구현하여 보안 수준 강화
• 클래스 모듈 내 알림 기능을 개발하고 Firebase Cloud Messaging(FCM)을 연동하여 모바일 푸시 알림 지원
• 통합회원 모듈에서 객관식·주관식·파일 업로드형 설문 기능을 개발하고 외부 기관 및 내부 클래스 모듈에 API 형태로 제공
• 시스템 전반 유지보수 및 고도화 작업에 지속적으로 참여
문제/해결/성과

문제

모바일 푸시 연동 중 Firebase SDK 인증 설정 오류로 알림 전송이 실패하는 문제가 발생했습니다.

해결

• 서비스 계정 키(JSON)를 프로젝트 리소스에 적용하고, `FirebaseApp.initializeApp()`을 안전하게 단일 초기화하도록 초기화 로직을 개선하여 인증 문제를 해결했습니다.
• 초기화 과정과 관련 설정을 문서화해 이후 개발/운영 환경에서도 동일한 방식으로 적용하도록 했습니다.

성과

• 웹·모바일 간 실시간 알림 동기화를 달성하여 공지 전달 신뢰도를 회복했습니다.
• 외부기관 연동 설문 API를 안정화하여 시스템 활용도를 높였습니다.
기술 스택
Java, Spring, MyBatis, JSP, MariaDB/Oracle/MySQL, REST API, Swagger, Bcrypt, Firebase Cloud Messaging

경상남도교육청 미래교육원 2024.03 ~ (진행) · 4명

체험 프로그램 예약/연수/결제 운영 시스템
프로젝트 소개
경상남도교육청이 운영하는 ‘미래교육원’은 학생·교직원·학부모 대상의 체험 프로그램 예약, 연수 관리, 유료 결제까지 지원하는 통합 플랫폼입니다.
예약 신청부터 결제, 알림톡 발송, QR코드 기반 현장 참여 확인 및 퀴즈/점수 부여까지 교육 체험 전 과정을 디지털화하여 운영 효율성과 사용자 편의성을 높이고 있습니다.
주요 업무
• 체험 프로그램 예약/결제 트랜잭션 로직 설계 및 동시성 제어 구현
• 결제 모듈을 TOSS PG로 연동하고 결제 검증·콜백 처리 강화
• 예약 완료 및 취소 시 비즈톡 알림톡 발송 기능 개발
• QR코드 기반 퀴즈 점수 부여 및 현장 이벤트 처리 기능 개발
• Quartz를 활용한 휴면 계정 자동 삭제 구현
• 시스템 전반 유지보수 및 장애 대응 참여 중
문제/해결/성과

문제

피크 시간대 동시 예약 처리에서 좌석이 초과(오버부킹)되고 간헐적 데드락으로 예약 지연이 발생했습니다.

해결

• DB 행 잠금(`SELECT ... FOR UPDATE`)을 적용해 좌석을 동시 잠금하고, 데드락 발생 시 백오프(backoff) 기반으로 재시도(최대 N회)하도록 재시도 로직을 구현하여 트랜잭션을 안정적으로 재실행하도록 했습니다.

성과

• 오버부킹을 제거하여 동시예약 시 좌석 초과 0건을 달성했습니다.
• 데드락 발생 상황에서도 예약이 안정적으로 처리되도록 만들어 서비스 신뢰도를 높였습니다.
• TOSS 연동과 비즈톡 알림 도입으로 결제 및 예약 안내 UX를 개선했습니다.
기술 스택
Java, Spring, MyBatis, JSP, MariaDB, Quartz, TOSS PG, Biztalk

충북 통합 도서관 웹사이트 2024.01 ~ (진행) · 3명

도내 20개 교육도서관 통합 · 예약·대출·전자도서관
프로젝트 소개
충청북도 내 20개 교육도서관을 하나로 묶어, 도서 검색, 예약, 대출, 전자도서관 서비스를 제공하는 통합 사이트입니다.
사용자는 한 계정으로 모든 도서관을 이용할 수 있고, 체험 프로그램 예약과 도서관 서비스까지 통합된 환경에서 이용할 수 있습니다.
주요 업무
• 일일 도서관 체험 예약 캘린더 개발 (달력에서 날짜별 프로그램 조회·예약 가능)
• 회원·도서 예약/대출 기능 유지보수 및 개선
• 느린 SQL 쿼리 수정 및 검색 속도 개선
• 운영 중 발생하는 오류 해결 및 기능 추가
문제/해결/성과

문제

체험 프로그램 일정이 목록 형태로만 제공되어 사용자가 원하는 날짜의 예약 가능 여부를 빠르게 확인하기 어려웠습니다.

해결

• 캘린더 UI를 도입해 날짜별로 프로그램을 보여주고, 달력에서 바로 예약할 수 있게 기능을 구현했습니다.
• 자주 쓰이는 검색에 인덱스를 추가하고 중복 조회를 줄여 조회 속도를 개선했습니다.

성과

• 사용자가 달력에서 바로 예약할 수 있게 되어 사용 편의성이 크게 향상되었습니다.
• 월 1,000건 이상의 체험 예약을 안정적으로 처리하고, 검색 응답을 수초에서 1초 이내로 단축했습니다.
기술 스택
Java, Spring MVC, iBatis, JSP, OracleDB

알콩 플랫폼 2025.07 ~ (진행) · 6명

AI 기반 학습 통합 · SCAP 인증 대응
프로젝트 소개
AI 기반 학습 통합 플랫폼으로 학습 관리·콘텐츠·분석 기능을 제공하며, 현재 SCAP 보안인증을 준비하고 있습니다.

주요 업무
SCAP(보안인증) 대응을 위한 취약점 점검 및 패치 수행
CODE-RAY 를 이용한 내부 진단 → 취약점 수정 → 재검사 반복
시큐어 코딩(입력 검증·출력 이스케이프·CSRF 대응) 적용 및 공통 모듈 정리
감리 대응 및 보안 이행 관리

성과
정기 스캔과 조치로 현재 취약점 0건 유지 상태(감리 진행 중)
보안 조치 표준화로 패치·검증 속도 향상 및 유지보수 부담 경감

나의 역할
취약점 진단·재현·패치 주도 및 감리 대응 창구 역할
보안 가이드 작성 및 팀 교육 지원

기술 스택
Java, Spring, MyBatis, JSP, MariaDB/Oracle/MySQL, 보안 스캐너
